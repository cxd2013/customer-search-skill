# 客户搜索模板参考

## 常用搜索关键词

### 眼镜/视光行业
- 视光中心 电话
- 眼镜店 联系方式
- 眼科医院 电话
- 眼镜城 联系电话

### 组合关键词
- 眼镜店 + 视光中心 + 联系方式 + 电话
- 配镜中心 + 联系方式
- 近视防控 + 联系电话

### 地域搜索
- 北京 视光中心 电话
- 上海 眼镜店 联系方式
- 广州 眼科医院 电话

## 搜索技巧

1. **使用 Bing 搜索**：避免搜狗的反爬虫验证
2. **添加"电话"关键词**：直接获取联系方式
3. **使用地图搜索**：切换到地图标签获取更详细信息
4. **滚动加载**：地图结果可能需要滚动加载更多

## 数据提取字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 客户名称 | 商户/企业名称 | 昕仁眼镜视光中心 |
| 地址 | 详细地址 | 通州区潞苑南大街1号院 |
| 城市 | 所在城市 | 北京市 |
| 联系电话 | 电话号码 | 13520096293 |

## 数据清洗规则

1. 去除电话号码中的空格：`135 2009 6293` → `13520096293`
2. 删除无电话记录：过滤联系电话为空的行
3. 统一格式：保持固话格式（如 `010-12345678`）
